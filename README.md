# Paris-Housing-Data-Analytics-Dashboard
This project analyzes Paris housing prices using Power BI. 
The dashboard explores pricing trends, property features, and comparisons 
using interactive visualizations.

This project focuses on data analytics and visualization. I worked on transforming data, creating measures, and building dashboards to extract insights. I use GitLab to manage and document my analytics projects.


## Tools Used
- Power BI
- Git / GitHub / GitLab

## Key Features
- Custom measures (Avg_Price_Purnima)
- Multiple visualizations: bar, line, donut, treemap, table
- Interactive filtering and insights
